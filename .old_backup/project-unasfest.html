Website UNAS FEST 2024

Platform digital resmi untuk festival universitas terbesar, UNAS FEST, yang melayani ribuan pengunjung.

Deskripsi Proyek

Sebagai Fullstack Developer di kepanitiaan, saya memegang peran kunci dalam pengembangan antarmuka (front-end) untuk website resmi UNAS FEST 2024. Proyek ini bertujuan untuk menjadi pusat informasi bagi seluruh rangkaian acara, termasuk seminar, webinar, dan berbagai kompetisi.

Tugas saya meliputi pembuatan komponen UI dari awal menggunakan TypeScript dan Tailwind CSS, merancang tata letak halaman yang responsif, dan memastikan fungsionalitas berjalan lancar. Saya berkolaborasi erat dengan tim desain dan back-end untuk mengintegrasikan data acara dan konten secara dinamis.

Tantangan & Solusi

Tantangan utama adalah menciptakan website yang stabil dan berperforma tinggi untuk menangani lonjakan trafik selama periode puncak festival. Solusinya, saya fokus pada optimasi komponen React, melakukan code splitting, dan melakukan pengujian serta debugging secara intensif untuk memastikan tidak ada bottleneck. Hasilnya, website dapat berjalan dengan optimal dan memberikan pengalaman pengguna yang baik bagi lebih dari 10.000 pengunjung.
