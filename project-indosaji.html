Indosaji E-commerce

Platform e-commerce makanan yang dibangun secara fullstack menggunakan MERN Stack.

Deskripsi Proyek

Indosaji adalah proyek aplikasi web e-commerce yang dirancang untuk penjualan produk makanan secara online. Aplikasi ini memiliki fitur lengkap mulai dari pemilihan produk, sistem keranjang belanja, hingga proses pembayaran.

Saya bertanggung jawab penuh sebagai Fullstack Developer, mulai dari merancang skema database di MongoDB, membangun RESTful API dengan Node.js dan Express, hingga mengembangkan antarmuka pengguna (UI) yang interaktif menggunakan React.

Tantangan & Solusi

Salah satu tantangan utama adalah mengelola state aplikasi secara efisien, terutama pada fitur keranjang belanja. Solusinya adalah dengan mengimplementasikan React Context API untuk manajemen state global, sehingga data keranjang belanja dapat diakses dan diperbarui secara konsisten di seluruh komponen aplikasi.

Teknologi

React Node.js Express.js MongoDB REST API CSS
