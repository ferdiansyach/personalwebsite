## ✨ Portfolio Website

This repository contains the source code for my personal portfolio website. It has been built to be highly performant, responsive, and visually engaging with a modern aesthetic, glassmorphism UI elements, and scroll reveal animations.

### 🌟 Features

- **Modern UI/UX**: Premium dark theme with a vibrant gradient mesh and glassmorphism elements.
- **Bilingual Interface**: Seamlessly switch between Indonesian (ID) and English (EN) languages.
- **Responsive Design**: Flawless layout on mobile, tablet, and desktop devices.
- **Interactive Animations**: Typing effects, hover interactions, scroll-based stagger reveals, and smooth scrolling.
- **Dynamic Projects Showcase**: Detailed project pages (`Indosaji E-commerce`, `Smart Meter Analysis`, `Anambas Tourism`, `UNAS FEST`, `HIMASI`) featuring custom lightbox image galleries.
- **Scroll-to-Top**: Convenient floating button to navigate back to the top of the page.

---

## 💻 Tech Stack

- **HTML5**: Semantic and accessible page structure.
- **Tailwind CSS (via CDN)**: Rapid and highly-customizable atomic styling.
- **Vanilla JavaScript**: Lightweight and fast DOM manipulation, state management for themes and languages, and intersection observers for animations.
- **CSS3 Variables & Animations**: Custom properties for consistent theming and smooth keyframe animations (gradient mesh, fade-ins).

## 🛠️ How to Run Locally

Since this is a purely static website built with HTML, CSS, and JS (No build tools or heavy node modules required), running it is very simple:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ferdiansyach/personalwebsite.git
   ```
2. **Navigate into the directory:**
   ```bash
   cd personalwebsite
   ```
3. **Start a local local server:**
   - Using Python:
     ```bash
     python -m http.server 8080
     ```
   - *Or* using Node (http-server):
     ```bash
     npx http-server -p 8080
     ```
   - *Or* simply open `index.html` in your browser using the VS Code Live Server extension.
4. **View the site:** Open `http://localhost:8080` in your web browser.
