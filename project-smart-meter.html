Analisis Smart Meter dengan AI

Proyek analisis data dan machine learning untuk mendeteksi anomali konsumsi energi.

Deskripsi Proyek

Proyek ini merupakan bagian dari magang saya, di mana saya melakukan analisis tekno-ekonomi untuk efisiensi energi di gedung pintar. Tujuannya adalah untuk membangun model yang dapat memprediksi konsumsi energi dan menandai penggunaan yang tidak normal (anomali).

Saya bertanggung jawab atas seluruh alur kerja data science, mulai dari pengumpulan data, pembersihan, pra-pemrosesan (termasuk interpolasi dan integrasi data cuaca), hingga membangun dan mengevaluasi beberapa model machine learning seperti Gradient Boosting dan LSTM.

Tantangan & Solusi

Tantangan utamanya adalah menangani data time-series yang memiliki banyak nilai hilang (*missing values*). Saya mengatasinya dengan menerapkan teknik interpolasi linear untuk mengisi kekosongan data secara logis. Selain itu, pemilihan fitur yang relevan dari puluhan variabel juga krusial, yang saya selesaikan dengan analisis korelasi untuk mengurangi multikolinearitas dan meningkatkan akurasi model.
