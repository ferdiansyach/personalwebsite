Website UNAS FEST

Website resmi untuk acara festival terbesar di Universitas Nasional.

Deskripsi Proyek

Sebagai bagian dari kepanitiaan, saya bertugas sebagai Fullstack Developer dengan fokus utama pada front-end untuk website resmi UNAS FEST. Website ini berfungsi sebagai pusat informasi utama untuk seluruh rangkaian acara, mulai dari jadwal, pendaftaran, hingga galeri.

Saya bertanggung jawab untuk membuat komponen website dari awal, menerjemahkan desain UI/UX menjadi kode, dan memastikan website dapat diakses dengan baik di berbagai perangkat (responsif). Selain itu, saya juga terlibat aktif dalam proses testing dan debugging untuk menjamin stabilitas website selama periode acara yang padat pengunjung.

Tantangan & Solusi

Tantangan utamanya adalah memastikan performa website tetap optimal meskipun diakses oleh ribuan pengunjung secara bersamaan selama puncak acara. Solusinya adalah dengan melakukan optimasi pada aset gambar, menerapkan *lazy loading*, dan memastikan kode TypeScript yang ditulis efisien dan tidak menyebabkan *bottleneck* pada rendering.

Teknologi

TypeScript React Tailwind CSS Fullstack Debugging
